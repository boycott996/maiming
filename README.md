## 介绍

脉命网,旨在收集996的公司名单,部门,职位信息,让有志升职加薪的同学找到理想的奋斗公司
暂时只支持github第三方登陆,即将支持以太坊免注册登陆
本网站不存储您的个人信息,只搜集github user id以免滥用者攻击
